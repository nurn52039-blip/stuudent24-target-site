STUUDENT24 TARGET MINI-SITE

1) Установка:
   pip install -r requirements.txt

2) Запуск:
   python app.py

3) Открыть сайт:
   http://127.0.0.1:5000

4) Посмотреть заявки:
   http://127.0.0.1:5000/leads

5) Где сохраняются заявки:
   leads.csv

6) WhatsApp номерін ауыстыру:
   app.py ішінен мына жолды тап:
   whatsapp_number = "77783674384"
   Өз нөміріңді қой.
